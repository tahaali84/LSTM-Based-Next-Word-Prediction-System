# LSTM-Based Next Word Prediction

This project implements a Next Word Prediction system using Long Short-Term Memory (LSTM) networks. The model is trained on Shakespeare's *Hamlet* text corpus to learn language patterns and predict the most probable next word for a given sequence of words.


## Features

- Text preprocessing and tokenization
- N-gram sequence generation
- Word embedding using Keras Embedding layer
- Deep learning-based language modeling using LSTM
- Real-time next-word prediction
- Streamlit web interface

## Dataset

The model is trained on:

**Shakespeare's Hamlet**

The dataset is obtained from the NLTK Gutenberg Corpus.
